# 📊 Dashboard de Análise de Dados

> Relatório de testes por mês gerado com Python. Calcula taxa de aprovação, exibe tabela formatada no terminal e gera gráfico visual dos dados com Matplotlib.

---

## 📈 O que o projeto faz

- Lê dados de testes mensais
- Calcula a **taxa de aprovação** de cada mês
- Exibe uma **tabela formatada** no terminal com Pandas
- Gera um **gráfico de linha** visual com Matplotlib
- Exporta o relatório em imagem `.png`

## ▶️ Como executar

**Pré-requisitos:** Python 3

```bash
# Clone o repositório
git clone https://github.com/najuiamnc/analise-dados-python

# Instale as dependências
pip install pandas matplotlib

# Execute
python dashboard.py
```
